# React Movie App called Movie - Discovery
A movie discovery web app built with React, Vite, and Material UI, using data from The Movie Database (TMDB) API.

## Features
###  Core Functionality
- Browse Popular, Upcoming, and Trending movies  
- View movie details (overview, release date, rating, and cast)  
- Explore actor pages with biography and filmography  
- Search movies by title  
- Filter movies by genre and sort by release year (newest or oldest)
- 
###  User Interaction
- Add and remove favorites using React Context  
- Write and submit reviews with validation and rating  
- Receive confirmation via a snackbar alert after submission  

###  UI / UX Enhancements
- Responsive dark theme with Material UI styling  
- Card hover animations 
- Redesigned filter card with genre, search, and sorting controls  
- Updated header navigation and page layout for better flow  
- Fully responsive across screen sizes  

##  Pages
| Page | Description |
|------|--------------|
| **Home** | Discover popular movies with filtering and sorting |
| **Trending** | View trending movies of the week |
| **Upcoming** | See upcoming movie releases |
| **Favorites** | Manage saved favorite movies |
| **Search** | Search TMDB for movies by name |
| **Actor Details** | View actor profiles and filmography |
| **Movie Details** | Full movie overview and cast list |
| **Write Review** | Add a review for a selected movie |
